# cellular-automaton
This is an implementation of 256 elementary cellular automatons: http://mathworld.wolfram.com/ElementaryCellularAutomaton.html

Example usage and output for rule 30:
```
./cellaut.pl 10 30 30
              ■
             ■■■
            ■■  ■
           ■■ ■■■■
          ■■  ■   ■
         ■■ ■■■■ ■■■
        ■■  ■    ■  ■
       ■■ ■■■■  ■■■■■■
      ■■  ■   ■■■     ■
     ■■ ■■■■ ■■  ■   ■■■
    ■■  ■    ■ ■■■■ ■■  ■
```

./cellaut.pl 10 30 110
